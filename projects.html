The Prospector

Hackathon Pitch

The Prospector centralizes event discovery by scraping public communities and transforming unstructured text into a clean, visual dashboard of actionable events — find events, save "wishes", and explore connections.

Key Features

  • Event discovery with keyword & location search
  • Interactive dashboard showing event cards (name, date, description)
  • AI-powered extraction (Gemini) to convert unstructured content into structured data
  • Persistent "Wishes" stored in MongoDB
  • Optional network graph visualization (D3.js)

Tech Stack & Quick Start

Next.js Node.js MongoDB Gemini API D3.js

Quick Start: clone, npm install, set MONGODB_URI, SESSION_SECRET, GEMINI_API_KEY, then npm run dev.
